## 数据内容

每日更新包含：
- **原始 JSON 数据** (机器可读，推荐自动化使用)
- **RSS XML** (结构化数据，含自定义 bi: 命名空间)
- HTML 报告 (人类可读，可选)

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| region | string | 区域名称 |
| dau | number | 日活跃用户数 |
| arpu | number | 每用户平均收入 (USD) |
| currency | string | 货币单位 |
| dateRange | string | 数据覆盖日期范围 |

## 数据范围
